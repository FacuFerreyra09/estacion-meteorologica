# 🌦️ Estación Meteorológica con ESP32

Proyecto escolar que mide temperatura, humedad y luz usando sensores.

## 🔧 Componentes
- ESP32
- DHT11
- LDR
- Resistencias

## 📊 Funciones
- Lectura de temperatura
- Lectura de humedad
- Medición de luz
- Envío por Serial Monitor

## 🚀 Cómo usar
1. Abrir Arduino IDE
2. Instalar librería DHT
3. Cargar `main.ino`
4. Abrir monitor serie
